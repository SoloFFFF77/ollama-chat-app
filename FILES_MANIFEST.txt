FOR TERMUX USERS (Recommended):
  1. Read: START_HERE.md (2 min)
  2. Read: QUICK_START.md (5 min)

  4. Wait: ~10 minutes for installation
  5. Start: ./start-all.sh
  6. Open: http://localhost:3000

FOR PC/LINUX TESTERS:
  1. Download Ollama: https://ollama.ai
  2. Read: QUICK_START.md section "For Local Testing"
  3. npm install
  4. node server.js
  5. Open: http://localhost:3000

FOR GITHUB UPLOAD:
  1. Read: GITHUB_SETUP.md
  2. Create repository
  3. Push files
  4. Share URL with others

FOR DOCKER DEPLOYMENT:
  1. Install Docker
  2. docker-compose up
  3. Open: http://localhost:3000

📋 QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════

DEPENDENCIES INSTALLED BY SCRIPT:
  - Node.js 14+ (or newer)
  - npm (Node package manager)
  - Ollama (local LLM framework)
  - Express.js 4.18.2 (via npm)
  - Axios 1.6.2 (via npm)
  - CORS 2.8.5 (via npm)

PORTS USED:
  - 3000 (Chat application) - configurable
  - 11434 (Ollama API) - internal

MODELS INCLUDED:
  - None (downloaded separately)
  Recommended: Mistral (~5GB)

STORAGE REQUIREMENTS:
  - Application: ~50MB
  - Models: 500MB - 5GB each
  - Total: Depends on models

RAM REQUIREMENTS:
  - Minimum: 2GB
  - Recommended: 4GB+
  - Per model: 2-8GB
