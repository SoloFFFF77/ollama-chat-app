❓ COMMON QUESTIONS (QUICK ANSWERS)
═══════════════════════════════════════════════════════════════════════════

Q: Can I run this on my phone?
A: Yes! Install Termux from F-Droid, then run the install script.

Q: Do I need internet?
A: Only for setup and downloading models. Then it's 100% offline.

Q: Which model should I use?
A: Mistral is best. ~5GB download, fast responses, good quality.

Q: Can I use it with friends?
A: Yes, if they're on the same network: http://<your-ip>:3000

Q: How do I backup my chats?
A: Click "Download Chat" button. Saves as JSON file.

Q: Is my data private?
A: Completely. Everything stays on your device.

Q: Can I customize it?
A: Yes! Full source code included. Edit as needed.

Q: How much storage?
A: Application ~100MB, models 500MB-5GB each. Depends on models.

Q: What if something breaks?
A: Kill processes and restart. Or reinstall if needed.

Q: Can I share it?
A: Yes! Push to GitHub, share the link. Others can install easily.
